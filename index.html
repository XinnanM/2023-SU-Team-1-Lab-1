Rules explained:
  • Pick 3 cards each time that may make a set.
  • Player will call SET by pressing key specified and pick the cards.
  • Clicking Hint button could reveal one card that could make the SET.
  • Be sure the SET you are picking is correct! Wrong SET or Running out of turn time would result in penalty
  • Once the game starts, try to find as many set as possible.
  • It is guaranteed to have SET on board. GOOD LUCK!
  • A SET must have SAME or DIFFERENT color AND;
  • A SET must have SAME or DIFFERENT shape AND;
  • A SET must have SAME or DIFFERENT shading AND;
  • A SET must have SAME or DIFFERENT number of shapes.
